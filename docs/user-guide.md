# 用户说明书

## 文字编辑

- 中文、英文、数字、emoji 和常见标点会尽量按词组/完整符号换行，减少英文单词、中文词组和标点被拆开的情况。
- 选中文字后使用「背景」，会给选中的文字加矩形文字背景。
- 没有选中文字时使用「背景」，会给整个文字框加矩形背景。
- 「阴影」用于柔和投影或发光效果；「背景」用于实心文字底色，两者用途不同。
- B / I / U 和对齐按钮在同一行，适合快速调整文字样式和排版。

## 文字框调整

- 拖动文字框右侧把手时，只调整宽度。
- 拖动文字框上下把手时，只调整高度。
- 拖动四角把手时，同时调整宽度和高度。
- 如果想避免文字位置变化，优先拖对应方向的边把手，而不是四角把手。

## 背景图与裁剪

- 选择「跟随图片」时，画布预览和导出会尽量保持同一张完整图片范围。
- 使用自由裁剪尺寸时，背景图会按覆盖画布的方式显示，导出时也会按同样方式生成。
- 如果画布是竖版，但背景图宽高不够，导出会以当前画布背景显示为准；建议先在画布中确认背景是否铺满、是否有留白。

## 橡皮擦

- 「轻触开启」用于打开擦除模式。
- 开启后在画布上滑动即可擦除背景图对应区域。
- 再次轻触可退出擦除模式。

## 导出

- 导出前建议先确认画布预览里的背景图、文字框背景、阴影和裁剪位置。
- 当前导出会尽量和画布预览保持一致，尤其是背景图跟随图片、自由裁剪和普通横排文字。
- 竖排文字、纹理文字、特殊字体首次加载时，仍建议等待预览稳定后再导出。
