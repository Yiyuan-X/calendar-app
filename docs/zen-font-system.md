## 2. 字体分类

- 禅修正文：禅意、空灵
- 高级长文：高级、经书
- 金句标题：文人、诗意、国风
- 柔和治愈：温柔、佛系
- 现代界面：现代、极简
- 书法封面：拂尘、国风
- 扩展清雅：春风、清楷、圆融、书卷、寺院、素净、古楷、端雅、明净
- 英文情绪：禅意英文、高级杂志、极简海报、东方文艺、修心英文、山水英文

## 3. Font Token

微信小程序使用 `page` 变量承载 token，语义等同于 Web 的 `:root`。

```css
page {
  --font-main: 'CardLXGWWenKai', 'PingFang SC', 'Noto Sans SC', sans-serif;
  --font-serif: 'CardSourceHanSerifSC', 'Source Han Serif SC', 'Songti SC', serif;
  --font-title: 'CardMaShanZheng', 'CardXiaXingKai', 'Kaiti SC', cursive;
  --font-zen: 'CardLXGWWenKai', 'CardLXGWWenKaiGB', 'Kaiti SC', serif;
  --font-ui: 'CardSmileySans', 'PingFang SC', 'Noto Sans SC', sans-serif;
  --font-handwrite: 'CardYouZai', 'CardQiuHongKai', 'Kaiti SC', cursive;
  --font-poetry: 'CardJiangXiZhuoKai', 'CardQiuHongKai', 'Kaiti SC', serif;
}
```

## 4. 推荐 Font Family

- 正文：`var(--font-main)`
- 古籍/节气：`var(--font-serif)`
- 标题/封面：`var(--font-title)`
- 留白禅意：`var(--font-zen)`
- UI 控件：`var(--font-ui)`
- 手写语录：`var(--font-handwrite)`
- 文人金句：`var(--font-poetry)`

## 5. 字重

- UI 导航：600-700
- 正文：400
- 日签正文：400-500
- 标题：500-700
- 经书长文：400，避免过粗
- 毛笔标题：400，靠字号和留白建立气势

## 6. 行距

- UI：1.45-1.6
- 日签正文：1.65-1.85
- 经书长文：1.8-2.0
- 标题：1.2-1.35
- 竖排或短句：1.7-2.2

## 7. 留白规范

- 卡片内边距：32-56rpx
- 标题到正文：28-44rpx
- 正文段间距：20-32rpx
- 画布边缘安全区：不少于 8%
- 金句每行建议 9-13 个汉字，避免满铺

## 8. 字体加载策略

- 菜单只显示气质标签。
- 点击字体时再调用 `wx.loadFontFace`。
- 导出前由 renderer 收集设计中实际用到的字体并预加载。
- 未加载成功时回退到系统安全字体。
- 大体积 CJK 字体不进入主包，优先使用 CDN 或分包。

## 9. 减少包体积

- 只保留 8-12 个字体入口。
- 大字体使用子集化文件。
- 只打包常用汉字、标点、数字、英文。
- Source Han Serif 类字体优先用子集或 CDN，不放完整 OTF。
- 低频字体按需下载，下载成功后缓存。

## 10. 自动高级感

- 字体少而精，不堆书法。
- 正文多用禅意/高级/空灵，标题才用拂尘/国风。
- 暖白、米色、淡墨色作为主背景。
- 降低饱和度，减少强对比描边。
- 字距微放，行距放松，保留呼吸感。

## 11. AI 自动推荐字体

建议按文案语义做规则映射：

- 含“慈悲、修心、放下、当下”：禅意
- 含“心经、金刚、经、偈”：经书
- 含“山、水、云、月、空”：空灵
- 含“愿、温柔、陪伴”：温柔
- 短标题或封面：拂尘/国风
- 长文说明：高级
- UI 辅助文案：现代

## 12. 页面使用建议

- 禅意做图编辑器：字体菜单使用气质标签。
- 日签模板：默认禅意。
- 修心长文：高级或禅意。
- 节气内容：经书或高级。
- 国风海报标题：国风、拂尘。
- 空境留白图：空灵。
- 按钮、导航、Tab：现代。

## 13. 东方禅意美学判断标准

- 一张图只使用 1-2 种字体。
- 标题有气势，正文要克制。
- 字号层级明确，避免满屏书法。
- 背景低饱和，文字略带墨色。
- 留白多于装饰。
- 字体选择表达情绪，而不是炫技。
